> [!note] Signs you will need a Chi-square test
>- conduct a significance test
>- given data from across 3 or more categories
>OR
>- given a two way table with two categorical variables and multiple categories for each.
# Three Types of Chi-Square Tests
- _the_ _**chi-square test for goodness of fit**_ 
	- (for a distribution of proportions of one categorical variable in a population)
- _the_ _**chi-square test for independence**_ 
	- (for associations between categorical variables within a single population)
- _the_ _**chi-square test for homogeneity**_ 
	- (for comparing distributions of a categorical variable across populations or treatments).
### Conditions
- Randomness:
 	sample was taken randomly or treatments were assigned **randomly** in an experiment.
- **Large Counts**:  
  	All expected counts are at least 5 
## Chi Square test for goodness of fit
observed data matches expected distribution
start w 1 categorical variable with multiple categories or levels
- We have an expected distribution of frequencies for each category based on some theoretical or assumed distribution. This could be a specific hypothesis or an expected distribution based on historical data or theoretical expectations.
- tell you expected distribution
### null
observed frequencies match the expected
### alternative
observed frequencies in at least one category does not match the expected
## 3 condition test
1. ﻿﻿) The sample was random
2. ﻿﻿) The sample size is less than 10% of the population to assume independence
3. ﻿﻿) There are 5 or more expected counts in each category.
4. ﻿﻿Chi-square test for independence (or test for association)
	- two way table
5. ﻿﻿Chi Square Test for Homogeneity
	- two way table for categorical variables
