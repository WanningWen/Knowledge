# Definition #definition  
A **discrete** random variable is a random variable (RV) whose possible values either:  
- Constitute a **finite set**, or  
- Can be listed in an **infinite sequence** with a first element, second element, and so on (i.e., *countably infinite*).  

>[!info] To study basic properties of discrete rv’s, only the tools of discrete mathematics—summation and differences—are required.
