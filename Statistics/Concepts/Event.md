# Definition:
An event is any collection (subset) of outcomes contained in the sample space $S$ . An event is **simple** if it consists of exactly one outcome and **compound** if it consists of more than one outcome.
- Simple event
- Compound event
## Example:
Consider an experiment in which each of three vehicles taking a particular freeway exit turns left $(L)$ or right $(R)$ at the end of the exit ramp. The eight possible outcomes that comprise the sample space are:
$$

{ LLL, RLL, LRL, LLR, LRR, RLR, RRL, RRR }.

$$
Thus, there are eight simple events, among which are:
$$

E_1 = { LLL } \quad \text{and} \quad E_5 = { LRR }.

$$
**Compound Events**
• **Event** **:** The event that exactly one of the three vehicles turns right:
$$

A = { RLL, LRL, LLR }.

$$
• **Event** **:** The event that at most one of the vehicles turns right:
$$
B = { LLL, RLL, LRL, LLR }.
$$
• **Event** **:** The event that all three vehicles turn in the same direction:$$
