# Definition #definition 
A random variable is continuous if both of the following apply:  
1. Its set of possible values consists of either:  
   - All numbers in a single interval on the number line (possibly infinite in extent, e.g., from $-\infty$ to $\infty$), or  
   - All numbers in a disjoint union of intervals (e.g., $[0, 10] \cup [20, 30]$).  
2. No possible value of the variable has a positive probability, that is:     $$
   P(X = c) = 0 \quad \text{for any possible value } c.
   $$
>[!info] The study of continuous variables requires the continuous mathematics of the calculus—integrals and derivatives.
